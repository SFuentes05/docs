# Sistema de Validación de Consumos Paso Rápido

Esta herramienta permite validar automáticamente los consumos de peajes exportados desde la aplicación Paso Rápido del gobierno dominicano, utilizando datos de telemetría GPS de ERM Telematics para detectar fraudes y optimizar costos operativos.

![Dashboard principal del sistema de validación](/images/main_dashboard.png)

<Note>
Este módulo de validación está diseñado para empresas que manejan flotas de vehículos registradas en ERM Telematics y necesitan validar automáticamente los cargos de peajes de Paso Rápido contra datos reales de ubicación GPS.
</Note>

## Flujo de Trabajo del Sistema

<Tabs>
<Tab title="1. Exportación desde Paso Rápido">
  **Extracción de Datos**: El personal de logística exporta los consumos de peajes desde la aplicación oficial Paso Rápido del gobierno dominicano en formato CSV.
  
  - Datos incluyen: fecha, tag, estación, monto, tipo de vehículo
  - Exportación desde el portal oficial del gobierno
  - Archivos preparados para importación automática
</Tab>

<Tab title="2. Importación y Procesamiento">
  **Carga de Datos**: Los archivos CSV se importan en el sistema y se procesan automáticamente.
  
  - Vinculación automática con datos ERM Telematics
  - Uso de `tag_assignments` como tabla puente
  - Sincronización con vehículos, proyectos, eventos y viajes
</Tab>

<Tab title="3. Validación Automática">
  **Validación GPS**: Compara ubicación del vehículo en ERM con la estación de peaje al momento del cobro.
  
  - Tolerancia de tiempo: ±1 minuto por defecto
  - Tolerancia de distancia: 500 metros por defecto
  - Detección automática de anomalías y duplicados
</Tab>

<Tab title="4. Análisis y Reportes">
  **Generación de Reportes**: Produce análisis detallados con resultados de validación para revisión.
  
  - Reportes ejecutivos con estadísticas
  - Exportación a Excel con filtros aplicados
  - Identificación de patrones sospechosos
</Tab>
</Tabs>

## Integración de Sistemas

<Steps>
<Step title="Extracción desde Paso Rápido Gobierno">
  El personal de logística exporta datos desde la aplicación oficial:
  - **Consumos de Peajes**: CSV con todos los cargos del período
  - **Información de Tags**: Números de tag y estaciones
  - **Datos de Facturación**: Montos, fechas y tipos de vehículo
  - **Exportación Manual**: Proceso controlado por el usuario

  <Check>
  Los datos provienen directamente de la fuente oficial del gobierno dominicano
  </Check>
</Step>

<Step title="Conexión con ERM Telematics">
  El sistema conecta automáticamente con los datos de ERM:
  - **Vehículos y Proyectos**: Importados desde ERM Karma API
  - **Eventos GPS**: Telemetría en tiempo real de ubicaciones
  - **Viajes**: Rutas completas con inicio y fin
  - **Tabla Puente**: `tag_assignments` vincula Paso Rápido con ERM

  <Warning>
  La validación depende de la correcta asignación de tags a vehículos ERM
  </Warning>
</Step>

<Step title="Validación Automática">
  El sistema ejecuta validaciones cruzando ambas fuentes:
  1. **GPS**: Confirma que el vehículo ERM estuvo en la estación
  2. **Duplicados**: Identifica cargos repetidos en Paso Rápido
  3. **Categoría**: Verifica montos contra tarifas oficiales
  4. **Tags**: Valida estado activo al momento del cargo

  <Tip>
  Este módulo es parte de una aplicación más amplia que incluirá visualización de viajes
  </Tip>
</Step>
</Steps>
