# United Logistics

United Logistics es un **producto** de operación de flota: procedimientos definidos para peajes (Paso Rápido), combustible, telemática y reportes ejecutivos. No es una plataforma genérica — cada módulo tiene un flujo de trabajo pensado para el día a día del operador.

## Flujo típico del mes (Paso Rápido)

```mermaid
flowchart LR
  A[Asignar tags] --> B[Importar consumos]
  B --> C[Validar vs GPS]
  C --> D[Revisión manual]
  D --> E[Reporte PDF / Excel]
```

1. Mantenga al día las [asignaciones de tags](/paso-rapido/tags).
2. [Importe](/paso-rapido/importar-consumos) el archivo de peajes (solo registros PASE).
3. [Valide](/paso-rapido/validar-consumos) contra telemetría.
4. Resuelva casos dudosos con [revisión manual](/paso-rapido/revision-manual).
5. Genere el [reporte de consumos](/paso-rapido/reporte-consumos) o [exporte a Excel](/paso-rapido/exportar).
