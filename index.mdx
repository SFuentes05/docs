# Bienvenido a Paso Rápido

Paso Rápido es un sistema integral de validación y gestión de consumos de peajes que utiliza telemetría GPS de ERM Telematics para detectar fraudes y optimizar costos operativos.

<Note>
Este sistema está diseñado para empresas que manejan flotas de vehículos y necesitan validar automáticamente los cargos de peajes contra datos reales de ubicación GPS.
</Note>

## Conceptos Fundamentales

<Tabs>
<Tab title="Validación GPS">
  **Validación de Ubicación**: El sistema cruza cada cargo de peaje con datos GPS del vehículo para confirmar que realmente pasó por la estación en el momento indicado.
  
  - Tolerancia de tiempo: ±1 minuto por defecto
  - Tolerancia de distancia: 500 metros por defecto
  - Estados: Válido, Ubicación Inválida, Sin Telemetría
</Tab>

<Tab title="Detección de Duplicados">
  **Identificación de Dobles Cargos**: Detecta automáticamente cargos duplicados basándose en:
  
  - Mismo tag
  - Misma estación
  - Fecha y hora similares
  - Monto idéntico o similar
</Tab>

<Tab title="Validación de Categoría">
  **Verificación de Montos**: Compara el monto cobrado contra las tarifas oficiales según la categoría del vehículo:
  
  - Categorías 1-5 (Auto, Camioneta, Bus, Camión, etc.)
  - Detecta cobros menores (favorable) y mayores (desfavorable)
  - Calcula diferencias automáticamente
</Tab>

<Tab title="Estado de Tags">
  **Verificación de Tags**: Confirma que el tag estaba activo en el momento del cargo:
  
  - Estados: Válido, Inhabilitado
  - Histórico de cambios de estado
  - Asignaciones por período
</Tab>
</Tabs>

## Flujo Principal del Sistema

<Steps>
<Step title="Importación de Datos">
  Los datos se importan desde múltiples fuentes:
  - **Consumos de Peajes**: Archivos CSV con cargos
  - **Datos GPS**: Telemetría desde ERM Karma API
  - **Información de Vehículos**: Flota y proyectos
  - **Asignaciones de Tags**: Gestión manual de tags

  <Check>
  Todos los datos se organizan por organización para multi-tenencia
  </Check>
</Step>

<Step title="Proceso de Validación">
  El sistema ejecuta 4 tipos de validación automática:
  1. **GPS**: Confirma presencia del vehículo
  2. **Duplicados**: Identifica dobles cargos
  3. **Categoría**: Verifica montos correctos
  4. **Tags**: Valida estado activo

  <Warning>
  Las validaciones requieren datos GPS actualizados y asignaciones de tags correctas
  </Warning>
</Step>

<Step title="Revisión Manual">
  Los operadores pueden:
  - Revisar casos sospechosos
  - Aprobar o rechazar cargos
  - Agregar comentarios explicativos
  - Generar reportes detallados

  <Tip>
  Enfócate primero en los cargos con múltiples validaciones fallidas
  </Tip>
</Step>
</Steps>
