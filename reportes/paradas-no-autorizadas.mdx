# Reporte de paradas no autorizadas

Ruta: `/dashboard/reporte-paradas-no-autorizadas`  
Operación diaria de alertas: `/dashboard/alertas/paradas-no-autorizadas`

## Generar el reporte

1. Elija rango de fechas.
2. **Generar reporte**.
3. Revise historial con mapas.
4. Exporte PDF/Excel.

Para cerrar o autorizar alertas en vivo, use el módulo [Alertas](/alertas).
