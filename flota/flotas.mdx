# Flotas

Ruta: `/dashboard/flotas`  
(` /dashboard/proyectos` redirige aquí)

Las flotas agrupan vehículos. El acceso por organización se controla con asignación de flotas (antes “proyectos”).

## Operación básica

1. Abra **Flotas**.
2. Pestañas **Activas** / **Inactivas**.
3. **Nueva flota** → nombre e ID interno.
4. Edite, desactive/reactive o exporte CSV según necesidad.

Columnas: ID interno, Nombre, Vehículos, Estado.
