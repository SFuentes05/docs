# Alertas

## Paradas no autorizadas

Ruta: `/dashboard/alertas/paradas-no-autorizadas`

Pestañas: **Activas**, **Historial**, **Reporte**.

Acciones típicas sobre una parada:

- **Autorizar y crear POI** — convierte la parada en punto autorizado
- **Solo cerrar alerta** — cierra sin crear POI

Enlace al [reporte formal](/reportes/paradas-no-autorizadas).

## Configuración de alertas

Ruta: `/dashboard/alertas/configuracion`

Defina reglas: alcance (todos / vehículos / flotas), tipo de evento, notificaciones, activar/desactivar.
