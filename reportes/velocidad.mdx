# Reporte de velocidad

Ruta: `/dashboard/reporte-velocidad`

1. Filtre fechas, proyecto/flota, vehículos, límite y duración mínima.
2. **Generar reporte**.
3. Revise VMP por conductor, matriz por viaje y tabla de excesos.
4. **Exportar** PDF/Excel.

Existe también la vista legacy `/dashboard/excesos-velocidad` para consultas similares.
