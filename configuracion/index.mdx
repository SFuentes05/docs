# Configuración

Ruta: `/dashboard/configuracion`

Secciones típicas:

| Sección | Contenido |
|---------|-----------|
| **Organización** | Nombre, avatar, estadísticas |
| **Mi perfil** | Nombre, cargo, avatar |
| **Usuarios** | Invitar, roles, permisos — [detalle](/configuracion/usuarios) |
| **API** | Claves REST v1 — [detalle](/configuracion/api-keys) |

## Super Admin — Organizaciones

Ruta: `/dashboard/organizaciones`

- Crear / archivar organizaciones
- Activar módulos: Paso Rápido, Telematics, POIs, Combustible
- Detalle por org: usuarios, acceso a flotas (proyectos), módulos
