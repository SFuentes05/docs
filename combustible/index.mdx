# Combustible

Módulo para registrar y analizar cargas de combustible de la flota.

Ruta principal: `/dashboard/combustible`

## Procedimientos

| Qué necesita | Guía |
|--------------|------|
| Entender qué es una entrada | [Entradas](/combustible/entradas) |
| Crear productos (Diesel, etc.) | [Productos](/combustible/productos) |
| Crear proveedores / estaciones | [Proveedores](/combustible/proveedores) |
| Cargar muchas entradas | [Importar](/combustible/importar) |
| Informe de rendimiento | [Rendimiento](/combustible/reporte-rendimiento) |

## Pantalla principal

Botones típicos: **Rendimiento**, **Proveedores**, **Productos**, **Importar**, **Nueva entrada**.

KPIs: galones, costo, precio/gal promedio, rendimiento GPS/proveedor. Filtre por vehículo y fechas.
