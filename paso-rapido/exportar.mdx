# Exportar Excel

## Consumos

1. Vaya a `/paso-rapido/consumos`.
2. Aplique filtros al alcance deseado.
3. Pulse **Exportar Excel**.

El archivo incluye resumen (válidos, sospechas, sin validar, etc.) y detalle del conjunto filtrado: IDs, fechas, tags, estaciones, montos, estados de validación, revisión manual y comentarios.

## Tags

1. Vaya a `/paso-rapido/tags`.
2. Filtre si aplica.
3. Pulse **Exportar Excel**.

## Tips

- El export **respeta filtros**: use eso para enviar solo un propietario o solo sospechosos.
- Para presentaciones ejecutivas combine Excel + [PDF de consumos](/paso-rapido/reporte-consumos).
