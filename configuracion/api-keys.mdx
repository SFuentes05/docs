# Claves API

En **Configuración** → **API** (solo administradores de la organización).

## Crear una clave

1. Pulse **Nueva clave**.
2. Asigne un nombre descriptivo.
3. Copie el secreto completo **una sola vez** (`ulk_<prefix>_<secret>`).
4. Úselo como `Authorization: Bearer …` contra `https://unitedpetroleum.app/api/v1`.

Desde el mismo panel puede copiar la Base URL, ver el Organization ID, regenerar o revocar claves, y abrir la [documentación API](/api).

Detalle técnico: [API → Autenticación](/api/autenticacion).
