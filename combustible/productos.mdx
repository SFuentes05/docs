# Productos

Desde **Combustible** → **Productos** (también rutas legacy bajo Paso Rápido dashboard).

## Crear producto

1. Abra el modal/lista **Productos**.
2. Cree uno con: **nombre**, **tipo**, **precio/galón** por defecto, ID interno si aplica, activo/inactivo.
3. Guarde.

## Uso

Las [entradas](/combustible/entradas) y la [importación](/combustible/importar) referencian estos productos. Mantenga nombres estables (ej. `Diesel`) para que integraciones y archivos coincidan.
